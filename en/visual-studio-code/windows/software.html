Visual Studio Code is a fast, functional, and, most importantly, absolutely free code editor from Microsoft which offers the user all the necessary tools for development.

It features syntax highlighting for various programming languages, code autocompletion using IntelliSense technology, debugging right in the editor, full-featured support of Git and other SCMs, a full-featured terminal, and much more.

In addition, you can significantly extend the editor's functionality by using numerous plugins and add-ons, which not only make working with the code more convenient, but also allow you to personalize the application by changing the design theme, adding various icons, etc.

Key features of the application:

  • High working speed
  • Out-of-the-box support for a wide variety of programming languages (e.g. JavaScript, TypeScript, CSS, HTML, etc.). If there is a language support need to be added to the editor, the necessary add-on can be installed from the internal VS Code Marketplace repository.
  • Syntax highlighting, code autocompletion using IntelliSense technology, etc.
  • Built-in code debugger.
  • Support for Git and other version control systems which allows you to commit directly from the editor.
  • Built-in terminal.
  • Huge number of various plugins.
