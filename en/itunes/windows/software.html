iTunes is not just a player that allows you to listen your favorite songs and watch movies. This is a full-fledged tool for organizing your media library which offers the user all the most necessary tools (and even more!).

For example, it is possible to create "smart" playlists that are formed based on user preferences, synchronize data with iPod, iPhone, iPad and Apple TV, record and import songs from CD-ROMs in MP3, AAC, Apple Lossless, AIFF, WAV formats, edit ID3 tags and much more.

The application also provides access to the branded online store iTunes Store and allows you to organize joint listening to music from other music libraries in a local network using the DAAP protocol.

Key features of the app:

  • Play music, movies, podcasts.
  • Organization of a multimedia library, the ability to create playlists, smart playlists and folders.
  • Easy navigation through the media library.
  • The presence of the Genius function that analyzes the user's media library and offers songs and movies in the iTunes Store, based on their preferences.
  • Access to the branded online store iTunes Store, allowing you to buy music, movies, iOS apps and books.
  • Built-in multi-band equalizer.
  • Record and import tracks from CD-ROMs in MP3, AAC, Apple Lossless, AIFF, WAV formats.
  • Normalization of the volume level.
  • Editing ID3 tags.
  • Mini-player mode.
  • Availability of the iTunes Radio service.
  • Sound visualization.
  • The ability to listen to music from other music libraries in the local network via the DAAP protocol.
  • Sync with iPod, iPhone, iPad and Apple TV.
