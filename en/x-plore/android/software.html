X-plore – one of the best file managers for mobile platforms is now available for Android smartphones and tablets. In addition to standard operations with files and folders (copying, moving, deleting, etc.), the program has a built-in viewer of multimedia and text files, has a HEX editor, provides access to "cloud" storage of Google Drive, Dropbox, SugarSync, etc., can work with archives and much more.

Main features of X-plore:

  • Two-panel interface with a tree-like folder structure.
  • View hidden and system files and folders.
  • Built-in HEX editor.
  • Viewing multimedia and text files.
  • Working with archives.
  • Transfer files via Bluetooth, email, or send to an FTP server.
  • Access to various cloud services, including Google Drive, Dropbox, SugarSync, Box.net, SkyDrive, etc.
  • Support for FTP and FTPS servers.
  • View shared folders on other devices connected to the local network.
  • Create Picasa web albums, with the ability to upload, edit and delete photos.
  • Viewing SQLite database files (with the .db extension).
  • Editing file and folder attributes.
  • Support for "hot" keys.
  • Customizable keyboard shortcuts and much more.
